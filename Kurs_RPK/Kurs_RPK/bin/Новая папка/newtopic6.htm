Запрос задолженностей по группе

Работа с запросами ››

Для фильтрации списка по группе, пользователю необходимо выбрать соответствующий запрос из списка запросов, после чего на форме появится комбинированный список, содержащий список групп.

После выбора группы, пользователю необходимо нажать кнопку "Исполнить запрос", после этого, таблица будет отфильтрована и в ней остануться только задолженности студентов определенной группы, если они имеются.
