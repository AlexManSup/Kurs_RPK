Запрос задолженностей по дисциплине

Работа с запросами ››

Для фильтрации списка по дисциплине, пользователю необходимо выбрать соответствующий запрос из списка запросов, после чего на форме появится комбинированный список, содержащий список дисциплин.

После выбора дисциплины, пользователю необходимо нажать кнопку "Исполнить запрос", после этого, таблица будет отфильтрована и в ней остануться только задолженности студентов по определенной дисциплине, если они имеются.
